Deploying and testing Parse server

Time to get our Parse Server running in a droplet at DigitalOcean.

3.1. Preparing and deploying

The Azkfile.js is already prepared with the deployment system.
All you need is to do is to add the DEPLOY_API_TOKEN to our .env.

Let's get back to our terminal and type:

$ echo "DEPLOY_API_TOKEN=[YOUR_DIGITALOCEAN_TOKEN]" >> .env

Note: remember to replace [YOUR_DIGITALOCEAN_TOKEN] with your token.

Now use azk to deploy our Parse server to a droplet:

$ azk deploy
